## 1. Иерархия доверенных источников (российский контекст)

Источники ранжированы от наивысшего уровня доверия к низшему. Для каждого указано: что искать, как проверять актуальность.

### Уровень A — Первичные официальные источники (максимальное доверие)

#### A1. Росстат (Федеральная служба государственной статистики)
- **URL:** https://rosstat.gov.ru
- **Что искать:** демография, уровень жизни, занятость, здоровье населения, образование, социальная статистика по регионам и РФ в целом.
- **Проверка свежести:** на странице каждого показателя указан период (год, квартал). Российский статистический ежегодник обновляется ежегодно. Сравнивать дату публикации с текущей датой.
- **Базы данных Росстата:** https://rosstat.gov.ru/databases — витрина с детализацией по субъектам РФ.

#### A2. ЕМИСС (Единая межведомственная информационно-статистическая система)
- **URL:** https://fedstat.ru
- **Что искать:** официальная статистика всех федеральных органов исполнительной власти (не только Росстат). Показатели по здравоохранению, образованию, экологии, экономике.
- **Проверка свежести:** каждый показатель имеет дату последнего обновления. ЕМИСС определена единым источником достоверной и актуальной официальной статистической информации (Постановление Правительства РФ).
- **Открытые данные:** https://fedstat.ru/opendata

#### A3. Официальный интернет-портал правовой информации
- **URL:** http://pravo.gov.ru
- **Что искать:** тексты федеральных законов, указов Президента, постановлений Правительства в действующей редакции. Единственный официальный источник опубликования НПА.
- **Проверка свежести:** с 01.07.2022 тексты правовых актов с изменениями на портале являются официальными. Проверять наличие отметки «в действующей редакции».
- **Раздел «Законодательство России»:** http://pravo.gov.ru/ips/ — интегрированный полнотекстовый банк правовой информации.

#### A4. Профильные министерства и ведомства

| Ведомство | URL | Что искать |
|---|---|---|
| Минздрав РФ | https://minzdrav.gov.ru | Статистика здравоохранения, заболеваемость, смертность по нозологиям |
| Минтруд РФ | https://mintrud.gov.ru | Занятость, инвалидность, социальная защита, бедность |
| Минпросвещения РФ | https://edu.gov.ru | Статистика образования, дополнительное образование детей |
| Минобрнауки РФ | https://minobrnauki.gov.ru | Высшее образование, наука |
| МВД РФ | https://мвд.рф | Криминальная статистика, правонарушения несовершеннолетних |
| Минэкономразвития | https://economy.gov.ru | Социально-экономическое развитие, прогнозы |
| Минприроды | https://www.mnr.gov.ru | Экологическая статистика, особо охраняемые территории |
| Росмолодёжь | https://fadm.gov.ru | Молодежная политика, волонтёрство |

- **Проверка свежести:** искать дату публикации отчёта или статистического сборника. Ведомственные данные обычно публикуются с задержкой 6-12 месяцев.

#### A5. Федеральный реестр инвалидов (ФГИС ФРИ)
- **URL:** https://sfri.ru
- **Что искать:** статистика по инвалидности: численность, возрастная структура, распределение по группам, занятость инвалидов.
- **Проверка свежести:** данные обновляются ежемесячно. На сайте указана дата актуальности.

### Уровень B — Правовые справочные системы и официальные базы проектов

#### B1. КонсультантПлюс
- **URL:** https://consultant.ru
- **Что искать:** действующие редакции НПА, судебная практика, комментарии к законам. Удобен для быстрой проверки статуса закона (действует / утратил силу / изменён).
- **Проверка свежести:** система обновляется ежедневно. Указана дата актуальности базы.

#### B2. Гарант
- **URL:** https://garant.ru
- **Что искать:** аналогично КонсультантПлюс. Альтернативный источник для перекрёстной проверки НПА.
- **Проверка свежести:** ежедневное обновление.

#### B3. База проектов Фонда президентских грантов
- **URL:** https://президентскиегранты.рф (раздел «Проекты»)
- **Что искать:** все поддержанные проекты с 2017 года. Описания, бюджеты, результаты. Используется для анализа лучших практик обоснования и проверки ссылок на «ранее поддержанные проекты».
- **Дополнительные ресурсы:**
  - https://оценка.гранты.рф — экспертные оценки реализации проектов
  - https://гранты.рф (список 100 лучших) — образцовые примеры по направлениям
- **Проверка свежести:** данные обновляются после каждого конкурса (2 конкурса в год).

#### B4. Портал национальных проектов
- **URL:** https://национальныепроекты.рф
- **Что искать:** целевые показатели нацпроектов, текущие результаты, паспорта проектов. Важно для проверки ссылок на «цели нацпроектов» в заявках.
- **Госрасходы (нацпроекты):** https://spending.gov.ru/np/ — финансовые данные по нацпроектам.
- **Проверка свежести:** показатели обновляются ежеквартально.

### Уровень C — Академические и научные базы

#### C1. КиберЛенинка
- **URL:** https://cyberleninka.ru
- **Что искать:** полнотексты научных статей из российских рецензируемых журналов. Открытый доступ (Open Access, CC-BY). Полнотекстовый поиск по названиям, авторам, аннотациям, ключевым словам.
- **Проверка свежести:** каждая статья имеет год публикации. Фильтр по годам доступен в поиске.
- **Фильтры:** год публикации, рубрика по ГРНТИ, конкретный журнал.
- **Особенности:** поддерживает русскую морфологию. Более 3900 журналов, из них 2800+ в открытом доступе.

#### C2. eLibrary / РИНЦ (Российский индекс научного цитирования)
- **URL:** https://elibrary.ru
- **Что искать:** библиометрия, проверка существования статьи, автора, журнала. Более 39 млн публикаций и патентов. 4500+ журналов в РИНЦ.
- **Проверка свежести:** указан год публикации. Можно проверить, существует ли конкретная статья конкретного автора.
- **Ограничения:** веб-скрейпинг запрещён и ведёт к блокировке IP. Программный доступ — платный.

### Уровень D — Аналитические центры и мониторинговые системы

#### D1. «Если быть точным» (Фонд «Нужна помощь»)
- **URL:** https://tochno.st
- **Что искать:** визуализации и агрегированные данные по социальным проблемам в России — бедность, сиротство, доступность медицины, экология. Данные агрегированы из официальных источников.
- **Проверка свежести:** каждый датасет содержит ссылки на первоисточники. Проверять по первоисточнику.

#### D2. АСИ (Агентство социальной информации)
- **URL:** https://asi.org.ru
- **Что искать:** новости НКО-сектора, аналитика, лучшие практики. Полезен для контекста, но НЕ является первичным источником статистики.

#### D3. ВЦИОМ
- **URL:** https://wciom.ru
- **Что искать:** результаты социологических опросов, общественное мнение. Индексы социальных настроений.
- **Проверка свежести:** каждый опрос датирован. Указана выборка и методология.

#### D4. Реестр НКО Минюста
- **URL:** https://minjust.gov.ru/ru/activity/nko/ (раздел реестров)
- **Что искать:** проверка статуса НКО (действующая / ликвидирована), дата регистрации, организационно-правовая форма.

### Уровень E — Международные организации (с оговорками)

| Организация | URL | Когда использовать |
|---|---|---|
| ВОЗ (WHO) | https://who.int | Здоровье, эпидемиология — ТОЛЬКО с пометкой «международные данные» |
| ЮНИСЕФ | https://unicef.org | Дети — ТОЛЬКО с пометкой |
| Всемирный банк | https://data.worldbank.org | Экономика, бедность — ТОЛЬКО с пометкой |

> **ПРАВИЛО:** Международные данные применимы к РФ только при явном указании «по данным [организация], показатель для России составляет...». Нельзя экстраполировать глобальные средние на Россию.

## 2. Каталог паттернов галлюцинаций

Минимум 8 типов, каждый с примером из грантовой заявки.

### Паттерн 1: Вымышленные исследования с реальными авторами

- **Описание:** ИИ генерирует ссылку на работу реального учёного, которой не существует. Автор реален, название правдоподобно, но статья никогда не была опубликована.
- **Пример:** «Согласно исследованию Петровой Е.А. (2022) "Влияние цифровой среды на социализацию подростков в мегаполисах", 67% подростков испытывают трудности в офлайн-коммуникации».
- **Сигналы для детекции:**
  - Чрезмерно длинное и описательное название статьи
  - Отсутствие статьи в КиберЛенинке / eLibrary при поиске по автору + ключевым словам
  - Указан только автор и год, без журнала и DOI
- **Рекомендация:** Искать статью в eLibrary.ru по фамилии автора. Если не найдена — искать в КиберЛенинке по ключевым словам. Если нет — помечать как КРИТИЧЕСКАЯ ОШИБКА.

### Паттерн 2: Гиперточная статистика без источника

- **Описание:** ИИ генерирует убедительно конкретные цифры (с десятыми долями процента), которые не зафиксированы ни в одной открытой базе данных.
- **Пример:** «По данным Росстата, 31,4% детей в возрасте 7-14 лет в России не посещают учреждения дополнительного образования».
- **Сигналы для детекции:**
  - Точность до десятых (31,4% вместо «около 30%» или «каждый третий»)
  - Указание на Росстат или ЕМИСС, но показатель не находится в соответствующей базе
  - Необычно круглая или, наоборот, подозрительно точная цифра
- **Рекомендация:** Проверить цифру в ЕМИСС (fedstat.ru) и на сайте Росстата. Если показатель не найден — КРИТИЧЕСКАЯ ОШИБКА. Если найден похожий, но с другим значением — СУЩЕСТВЕННОЕ ЗАМЕЧАНИЕ.

### Паттерн 3: Несуществующие или устаревшие нормативные акты

- **Описание:** ИИ ссылается на закон в редакции, которая отменена или существенно изменена, либо на полностью несуществующий НПА.
- **Пример:** «В соответствии со ст. 15.2 Федерального закона от 28.12.2013 N 442-ФЗ "Об основах социального обслуживания граждан в РФ"...» (статья 15.2 не существует в данном законе).
- **Сигналы для детекции:**
  - Нетипичная нумерация статей (15.2, 28-а)
  - Несоответствие номера закона и названия
  - Ссылка на редакцию без указания даты редакции
- **Рекомендация:** Проверить через КонсультантПлюс (consultant.ru) или Гарант (garant.ru). Убедиться, что статья существует и закон в действующей редакции. Если закон утратил силу — КРИТИЧЕСКАЯ ОШИБКА.

### Паттерн 4: Географическое расширение (подмена масштаба)

- **Описание:** Региональная статистика (часто московская или петербургская) подаётся как общероссийская, или наоборот.
- **Пример:** «В России 78% людей с инвалидностью не имеют доступа к реабилитационным услугам» (на самом деле это данные по конкретному сельскому региону, а не по стране в целом).
- **Сигналы для детекции:**
  - Отсутствие указания на конкретный регион при использовании формулировки «в России»
  - Показатель резко отличается от известных средних значений
  - Ссылка на региональное исследование, но в тексте — общефедеральный вывод
- **Рекомендация:** Проверить, существует ли такой показатель на федеральном уровне (Росстат). Если есть — сравнить значения. Если расхождение >20% — СУЩЕСТВЕННОЕ ЗАМЕЧАНИЕ с требованием уточнить географию.

### Паттерн 5: Конвертация международной статистики на Россию

- **Описание:** Данные ВОЗ, Eurostat, ОЭСР или другой международной организации применяются к России без явной оговорки о том, что это международная, а не российская статистика.
- **Пример:** «Каждый четвёртый подросток в России страдает от кибербуллинга» (на самом деле это данные опроса ЮНИСЕФ по 30 странам, Россия в выборку не входила).
- **Сигналы для детекции:**
  - Формулировки типа «по данным ВОЗ, в России...» без прямой ссылки на страновой отчёт
  - Круглые цифры, характерные для глобальных оценок («каждый четвёртый», «каждый десятый»)
  - Показатели, для которых в РФ ведётся собственный учёт, но цифра не совпадает с Росстатом
- **Рекомендация:** Проверить, есть ли аналогичный российский показатель в Росстате/ЕМИСС. Если есть — использовать его. Если нет — допустима международная ссылка ТОЛЬКО с явной оговоркой «по международным данным» и указанием, что для РФ отдельных данных нет. Иначе — СУЩЕСТВЕННОЕ ЗАМЕЧАНИЕ.

### Паттерн 6: Смешение данных по временным срезам

- **Описание:** Цифры из разных лет объединены в одно утверждение, создавая ложную картину динамики или масштаба проблемы.
- **Пример:** «Число бездомных в России выросло с 3 млн человек до 4,5 млн» (первая цифра — оценка НКО за 2015 год, вторая — экстраполяция СМИ за 2023 год, методологии несопоставимы).
- **Сигналы для детекции:**
  - Утверждение о динамике («выросло», «снизилось», «увеличилось в N раз») без указания обоих временных точек
  - Две цифры рядом, но с разными (неуказанными) источниками
  - Сравнение абсолютных и относительных показателей
- **Рекомендация:** Требовать указание года для каждой цифры. Проверить, что обе цифры из одного источника с одной методологией. Если методологии различаются — СУЩЕСТВЕННОЕ ЗАМЕЧАНИЕ.

### Паттерн 7: Безликие экспертные утверждения

- **Описание:** Формулировки «эксперты отмечают», «по мнению специалистов», «исследования показывают» без указания конкретных лиц, организаций или публикаций.
- **Пример:** «По мнению ведущих экспертов в области детской психологии, раннее вмешательство снижает риск девиантного поведения на 40%».
- **Сигналы для детекции:**
  - Обобщённые субъекты: «эксперты», «специалисты», «учёные», «исследования»
  - Конкретная цифра при отсутствии конкретного источника
  - Невозможно установить, кто именно это заявил
- **Рекомендация:** Отнести к категории СРЕДНЕЕ ЗАМЕЧАНИЕ. Потребовать указать конкретного эксперта, организацию или публикацию. Если цифра сопровождает безликое утверждение — повысить до СУЩЕСТВЕННОГО ЗАМЕЧАНИЯ.

### Паттерн 8: Причинно-следственные спекуляции

- **Описание:** Корреляция или совпадение представлены как доказанная причинно-следственная связь.
- **Пример:** «Реализация программы наставничества в 2020-2022 гг. привела к снижению подростковой преступности в регионе на 15%» (снижение могло быть вызвано множеством факторов, а не конкретно данной программой).
- **Сигналы для детекции:**
  - Слова «привело к», «благодаря», «в результате» при описании сложных социальных процессов
  - Одна программа/инициатива приписывается масштабный социальный эффект
  - Отсутствие ссылки на оценку воздействия (impact evaluation) с контрольной группой
- **Рекомендация:** СРЕДНЕЕ ЗАМЕЧАНИЕ. Рекомендовать заменить на корреляционную формулировку: «в период реализации программы наблюдалось снижение...» вместо «программа привела к снижению...».

### Паттерн 9: Устаревшие данные без оговорки

- **Описание:** Данные старше 3-5 лет используются как актуальные, без указания на временной разрыв. Особенно критично для быстро меняющихся показателей (цифровизация, COVID-последствия).
- **Пример:** «По данным Росстата, 42% населения России пользуются порталом Госуслуг» (данные 2019 года; к 2024 году показатель превысил 75%).
- **Сигналы для детекции:**
  - Цифры, которые «кажутся устаревшими» (например, данные о цифровизации до пандемии)
  - Отсутствие года в ссылке на источник
  - Показатели, для которых есть известная динамика резких изменений (COVID, санкции, цифровизация)
- **Рекомендация:** Проверить, есть ли более свежие данные. Если данные старше 3 лет — СУЩЕСТВЕННОЕ ЗАМЕЧАНИЕ с указанием актуальных данных (если найдены). Если данные старше 5 лет — КРИТИЧЕСКАЯ ОШИБКА (если есть более свежие).

### Паттерн 10: Ложная атрибуция источника

- **Описание:** Цифра приписывается авторитетному источнику (Росстат, ВОЗ, Минздрав), но в действительности взята из СМИ, блога или другого неофициального источника, который сам не указал первоисточник.
- **Пример:** «По данным Минздрава, 60% россиян испытывают хронический стресс» (на самом деле это цитата из статьи в онлайн-СМИ, которое ссылалось на коммерческий опрос страховой компании).
- **Сигналы для детекции:**
  - Цифра не находится на официальном сайте указанного источника
  - Показатель не типичен для методологии указанного ведомства (Минздрав не измеряет «хронический стресс» как статистический показатель)
  - При поиске цифра обнаруживается только в СМИ, а не в первоисточнике
- **Рекомендация:** Искать цифру на официальном сайте указанного источника. Если не найдена — КРИТИЧЕСКАЯ ОШИБКА с пометкой «ложная атрибуция».

## 3. Правила приоритета источников

### 3.1. Российские vs международные в российском контексте

| Ситуация | Приоритет | Обоснование |
|---|---|---|
| Есть российский и международный показатель | **Российский** | Для заявки в российский фонд российские данные всегда приоритетнее |
| Российских данных нет, есть международные | **Международный с оговоркой** | Допустимо, но ОБЯЗАТЕЛЬНА формулировка: «по международным данным (источник, год), в отсутствие аналогичной российской статистики...» |
| Международная методология ВОЗ/ОЭСР | **Допустимо для контекста** | Можно использовать для сравнения, но основой должны быть российские данные |

### 3.2. Официальные vs неофициальные

| Тип источника | Уровень | Правило |
|---|---|---|
| Первичные официальные (Росстат, ЕМИСС, ведомства) | **Максимальный** | Принимаются без дополнительной верификации |
| Правовые базы (КонсультантПлюс, Гарант) | **Высокий** | Принимаются, но для критичных НПА желательна перекрёстная проверка через pravo.gov.ru |
| Рецензируемые журналы (РИНЦ, КиберЛенинка) | **Средний-Высокий** | Принимаются, если журнал входит в перечень ВАК или РИНЦ. Проверять импакт-фактор |
| Аналитические центры и НКО | **Средний** | Принимаются, если указана методология и первоисточник данных |
| СМИ (РБК, Ведомости, Коммерсант) | **Низкий** | Только как наводка на первоисточник. В заявке ссылаться на первоисточник, а не на СМИ |
| Блоги, Telegram-каналы, соцсети | **Не принимаются** | Не являются допустимым источником для грантовой заявки |

### 3.3. Актуальность данных

| Категория данных | Допустимый срок давности | Условия |
|---|---|---|
| Демография, базовая статистика | До 3 лет | Если нет более свежих, допустимо с оговоркой |
| Социально-экономические показатели | До 2 лет | Учитывать влияние COVID-19, санкций и других структурных изменений |
| Цифровизация, технологии | До 1 года | Быстро меняющиеся показатели; данные 2019 года и ранее — критически устаревшие |
| Нормативные акты | Только действующая редакция | Проверять через КонсультантПлюс или pravo.gov.ru |
| Показатели нацпроектов | До 1 года | С 2025 стартовали новые нацпроекты; данные по старым (2019-2024) требуют оговорки |

**Общее правило:** данные старше 5 лет не должны использоваться без специальной оговорки и обоснования, почему более свежие данные недоступны.

## 4. API и прямые ссылки для программной верификации

### 4.1. ЕМИСС / Fedstat

- **Веб-интерфейс:** https://fedstat.ru — поиск по показателям, фильтрация по годам и регионам.
- **Открытые данные:** https://fedstat.ru/opendata — наборы данных для скачивания.
- **Неофициальный API (R):** [fedstatAPIr](https://github.com/DenchPokepon/fedstatAPIr) — пакет для R, позволяет автоматизировать загрузку данных с fedstat.ru. Доступен через CRAN.
  - Ограничение: не более 500 тыс. строк за один запрос; ЕМИСС часто отвечает медленно.
- **Официального REST API нет.** Для программного доступа рекомендуется парсинг открытых данных.

### 4.2. Росстат

- **Основной сайт:** https://rosstat.gov.ru
- **Открытые данные Росстата:** https://rosstat.gov.ru/opendata — каталог наборов данных в машиночитаемых форматах.
- **Базы данных:** https://rosstat.gov.ru/databases — витрина статистических показателей.
- **Российский статистический ежегодник:** https://rosstat.gov.ru/folder/210/document/12994 — ежегодный сборник в PDF.
- **DaMIA API-Росстат:** https://damia.ru/apirs — коммерческий сервис для интеграции базы данных Росстата (автоматическая выгрузка финансовой отчётности).

### 4.3. КиберЛенинка

- **Поиск:** https://cyberleninka.ru/search — полнотекстовый поиск с поддержкой русской морфологии.
- **Прямая ссылка на статью:** `https://cyberleninka.ru/article/n/{slug}` — каждая статья имеет постоянный URL.
- **Официального API нет.** Для проверки существования статьи — использовать веб-поиск по названию и автору.
- **Рубрикатор ГРНТИ:** позволяет искать статьи по тематическим разделам.

### 4.4. eLibrary / РИНЦ

- **Поиск:** https://elibrary.ru — поиск по авторам, названиям, ключевым словам.
- **РИНЦ:** https://elibrary.ru/project_risc.asp — проверка наукометрических показателей автора.
- **API доступ:** платный, по запросу. Веб-скрейпинг запрещён (блокировка IP).
- **Для фактчекера:** использовать веб-интерфейс для проверки существования статьи/автора.

### 4.5. Правовые базы

- **КонсультантПлюс:** https://consultant.ru — бесплатный онлайн-доступ к части базы.
- **Гарант:** https://garant.ru — аналогично.
- **pravo.gov.ru:** http://pravo.gov.ru/ips/ — бесплатный полнотекстовый поиск по всему законодательству РФ.

### 4.6. Дополнительные ресурсы для верификации

| Ресурс | URL | Назначение |
|---|---|---|

## 5. Краткая памятка для системного промпта

### Презумпция ошибки
Любая статистика, исследование или ссылка на НПА «виновна, пока не доказана невиновность». Каждое числовое утверждение требует верификации через первоисточник.

### 5-шаговый процесс верификации
1. **Извлечение:** выписать все числовые утверждения, ссылки на исследования и НПА.
2. **Поиск источника:** для каждого утверждения найти первоисточник (Росстат, ЕМИСС, КиберЛенинка, pravo.gov.ru).
3. **Сверка:** сравнить цитируемые данные с оригиналом.
4. **Проверка актуальности:** убедиться, что данные / НПА актуальны.
5. **Классификация ошибок:** КРИТИЧЕСКАЯ / СУЩЕСТВЕННАЯ / СРЕДНЯЯ / НИЗКАЯ.

### Классификация ошибок

| Уровень | Что входит | Действие |
|---|---|---|
| **КРИТИЧЕСКАЯ** | Несуществующие исследования, ложная статистика, недействующие НПА, ложная атрибуция | Обязательно исправить перед подачей |
| **СУЩЕСТВЕННАЯ** | Устаревшие данные, географическая подмена, смешение временных срезов | Настоятельно рекомендуется исправить |
| **СРЕДНЯЯ** | Статистика без источника, безликие эксперты, причинно-следственные спекуляции | Рекомендуется уточнить |
| **НИЗКАЯ** | Стилистические неточности, избыточная детализация | На усмотрение автора |
